# 🦖 Chrome Dino Neuroevolution AI

An AI that learns to play the Chrome Dino game using **NeuroEvolution of Augmenting Topologies (NEAT)** and **Pygame**.

Instead of manually programming the dinosaur's behavior, a population of neural networks evolves over generations to achieve higher scores.

---

## 🚀 Features

- Population of **50 AI-controlled dinosaurs**
- Neural networks evolve using **NEAT-Python**
- Fitness-based natural selection
- Automatic mutation and crossover
- Real-time game visualization
- Generation and score tracking

---

## 🧠 How It Works

Each dinosaur is controlled by its own neural network.

For every generation:

1. 50 dinosaurs are spawned.
2. Each dinosaur tries to survive as long as possible.
3. Fitness is calculated based on performance.
4. The best-performing genomes reproduce.
5. Mutation creates new behaviors.
6. The next generation begins.

Over time, the population evolves and learns to avoid obstacles more effectively.
